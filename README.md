## 使用
### 安装

- Windows
- Ubuntu


### 写幻灯片
```
![](http://git.oschina.net/uploads/images/2016/1126/141745_d4135c13_856793.png) 
## A markdown slide application.

---

# Sample - List

Events:
- click
- blur
- ...

```

## 使用开源组件

- Markdown使用[**marked**](https://github.com/chjj/marked)解析
- 基于[**electron**](https://github.com/electron/electron)开发
